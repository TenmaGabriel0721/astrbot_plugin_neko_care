# 云猫娘养成 

一个功能完整的 AstrBot 插件，包含云养猫娘养成、独立的经济系统。
📕 本项目参考zerobot中的云养猫并通过ai进行重构和优化

## ✨ 功能特色

- 🐱 **猫娘养成**：许愿、喂养、互动、打工、自定义形象
- 🎁 **签到系统**：每日签到获取宝石 内置了横板和竖版两套好看的签到卡片渲染以及文字输出
- 💰 **经济系统**：钱包、打工赚钱、转账交易、商店道具、护理服务和打工地点解锁
- 📊 **排行榜**：钱包排行、猫娘羁绊排行
- 🎨 **高度可定制**：自定义签到背景、自定义猫娘图片、自定义每日一言语录
- 🧩 **插件拓展页**：在 AstrBot 插件 Pages 中实时调整经济、许愿、状态结算、喂食、打工地点、商店护理、互动和性格效果

## 📥 安装

1. 在 AstrBot 插件市场搜索「astrbot_plugin_neko_care」、「猫娘养成」
2. 点击安装并启用
3. 插件会自动创建必要的目录和默认资源

## 🎮 使用指令

### 签到与经济
- `签到` / `猫猫签到` - 每日签到领取宝石奖励，按配置返回图片或文字签到卡。
- `查看猫猫钱包` / `查看猫娘钱包` - 查询自己的宝石余额。
- `每日打工` - 主人每日打工一次，领取随机宝石收入。
- `钱包转账 数量 @用户` - 把自己的宝石转给指定用户，需要 @ 对方。
- `钱包排行榜` - 查看全局宝石余额排行榜 TOP 10。

### 猫娘养成
- `请赐我一只可爱猫娘吧` - 每日许愿遇见猫娘，成功后可确认收养或换一只候选（默认 80% 概率，3 次保底）。
- `带她回家` / `确认收养` - 确认收养当前候选猫娘。
- `换一只猫娘` / `换个形象` - 在待收养状态下切换到另一只候选猫娘。
- `成长档案` / `猫娘状态` / `猫猫状态` / `猫猫档案` - 查看自己的猫娘档案，包括阶段、亲密、饱食、心情、健康和精力。
- `喂猫` / `喂猫娘` / `喂猫猫` - 自动购买并喂食，恢复饱食并提升养成属性（饱食度低于 85 时可喂；饱食度约 48 小时从 100 降到 0）。
- `猫娘打工 [地点名|列表|解锁 地点名]` / `猫猫打工` / `打工` - 派猫娘打工，支持领取完成任务、指定地点、查看列表和解锁地点。
- `撸猫` / `逗猫` / `摸猫` / `rua猫` / `陪猫娘` / `贴贴猫娘` - 进行内置互动，消耗精力并提升心情、亲密和成长，受冷却与每日递减影响。
- `猫娘互动 动作名` / `猫猫互动 动作名` - 触发 Pages 中配置的自定义互动动作。
- `猫娘商店 [分类]` / `猫猫商店 [分类]` - 查看猫娘商店，可按分类浏览食物、礼物、道具和功能卡。
- `背包` / `猫娘背包` / `猫猫背包` - 查看自己的背包物品、数量和待生效加成。
- `购买 道具名 [数量]` / `购买道具 道具名 [数量]` - 消耗宝石购买商店道具，例如 `购买 小鱼干礼盒 3`。
- `使用 道具名` / `使用道具 道具名` - 使用背包道具；部分功能卡会在对应操作时自动消耗。
- `猫娘护理 [服务名]` / `猫猫护理 [服务名]` / `猫娘看病 [服务名]` - 查看或购买护理服务，例如看病、休息套餐、按摩护理。
- `猫娘改名 新名字` - 消耗 1 张改名卡修改猫娘名字。
- `更换猫娘形象` / `更换猫猫形象` - 更换猫娘图片；可直接带图，或先发指令后 2 分钟内发图，优先消耗形象更改卡。
- `羁绊排行榜` / `猫娘排行榜` / `猫猫排行榜` - 查看当前群的猫娘羁绊排行榜。
- `迁移猫娘到本群` / `猫娘迁移` - 把自己的猫娘登记到当前群，用于参与本群羁绊排行榜。

### 帮助
- `猫猫帮助` / `猫娘帮助` - 查看猫娘养成插件的完整帮助、指令分组和当前养成规则。

### 插件 Pages 配置

安装后可在 AstrBot 后台的插件拓展页打开 `neko-admin` 页面。页面会保存运行时配置到 `data/plugin_data/astrbot_plugin_neko_care/runtime_config.json`，保存后立即应用到当前插件实例。

可在页面中调整：

- 货币名、签到奖励、每日打工奖励和每日打工事件文本
- 许愿概率、保底次数、猫娘形象更换价格
- 饱食度、心情、健康、精力的按分钟线性结算参数和健康判定阈值
- 喂食阈值、低健康喂食倍率、食物池、食物价格和喂食收益
- 猫娘打工地点、耗时、奖励、解锁费用、阶段门槛、精力档位收益倍率、消耗、成长和亲密收益
- 商店道具、礼物收益、使用限制、下一次互动/打工加成、护理服务价格、恢复效果和冷却时间
- 互动动作、互动文本、阶段门槛、冷却、精力消耗、每日软上限、心情收益倍率和递减倍率
- 8 种猫娘性格的倍率效果，包括状态消耗/恢复、喂食收益、打工收益/消耗、互动收益/消耗

新增互动动作可通过 `猫娘互动 动作名` 或 `猫猫互动 动作名` 触发；内置的 `撸猫`、`逗猫` 等短命令仍保留。

## 🐱 养成体系

猫娘养成以「羁绊」为核心，不再鼓励单纯追求体重数值。喂食、互动、猫娘打工会提升不同属性：

| 属性 | 说明 |
|------|------|
| `成长进度` | 由隐藏成长值换算为当前阶段百分比，推动猫娘从初遇、熟悉、亲近一路成长到永伴 |
| `亲密等级` | 由隐藏亲密度换算为等级，等级越高升级所需亲密度越多 |
| `饱食度` | 按分钟线性下降，过低会影响健康，归零超过 7 天会离家出走；可用「命运的红线」召回 |
| `心情` | 按分钟线性下降，会受互动和打工影响；低心情会降低互动的成长/亲密收益，高心情会提高收益 |
| `健康` | 按分钟结算，长期饥饿或低心情会下降，照顾良好会缓慢恢复；低健康会降低喂食的附加收益 |
| `精力` | 按分钟线性恢复，猫娘打工和互动会消耗精力，高精力会提高打工报酬档位 |

### 经济消耗体系

宝石除了喂食和更换形象外，还可用于以下养成消耗：

- `猫娘商店` 中购买礼物、一次性恢复道具、下一次互动加成道具、下一次打工加成道具
- `改名卡` 和 `形象更改卡` 属于功能卡，收养猫娘后每位用户初始各赠送 1 张；改名会消耗改名卡，更换形象会优先消耗形象更改卡
- `猫娘护理` 中购买看病、休息、按摩、温泉等服务，用于恢复健康、心情、精力或饱食
- `猫娘打工 解锁 地点名` 解锁更高收益但有阶段门槛的打工地点

礼物有每日软上限，超过后亲密和成长收益递减；护理服务可以配置冷却，避免同一服务被连续刷收益。所有道具和护理服务都可以在 Pages 管理页中增删、改价格和改收益。

默认经济平衡按“日常收入覆盖喂食维护，商店、护理和打工解锁消耗长期盈余”设计：签到与每日打工合计平均约 155 宝石/天，饱食度 100 到 0 约 48 小时，普通喂食约需要每日 2 次维护。

成长阶段包括：初遇、熟悉、亲近、信赖、羁绊、灵契、永伴。喂食、互动、打工获得的成长值和亲密度仍按具体数值显示，状态档案中展示为成长进度和亲密等级。新猫娘初始体重为 40-90 斤范围内的小数存档，输出当前状态时显示整数，变动数值显示 1 位小数。旧版本的猫娘数据会在读取时自动补齐新字段，并将旧 kg 体重自动迁移为斤。

插件安装后会在 `data/plugins/astrbot_plugin_neko_care/assets/` 创建以下目录：

- `sign_backgrounds/` - 签到背景
- `catgirl_pool/` - 猫娘默认形象库
- `quotes.txt` - 一言语录文件

您可以添加自己的图片和语录，插件会随机选择使用。

## 📝 管理员指令

需在配置中添加 `extra_admin_ids`（AstrBot 自身管理员无需添加）：

- `管理员给 数量 @用户` - 给指定用户增加宝石，需要 @ 对方。
- `管理员扣 数量 @用户` - 扣除指定用户宝石，需要 @ 对方。
- `管理员查看 @用户` - 查看指定用户宝石余额，需要 @ 对方。
- `管理员清理缓存` - 按配置清理插件生成的缓存图片并返回释放空间。

## 🔧 常见问题


**Q: 如何更换签到背景？**  
A: 将图片放入对应目录，支持 jpg、png、webp 格式。

**Q: 猫娘数据是否群隔离？**  
A: 猫娘数据全局通用，但可通过「迁移猫娘到本群」登记到指定群，在该群羁绊排行榜中显示。
